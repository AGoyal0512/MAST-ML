# MAST-ml-private
MAST machine learning incubator - for direct scientific use - not intended for public yet

*   NOTE: Use Python 3

For usage on existing csv files, see for example `test/classtest_mastml/test.conf`::
    
    python /path/to/MASTML.py test.conf
